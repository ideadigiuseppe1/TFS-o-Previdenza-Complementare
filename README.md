# TFS-o-Previdenza-Complementare
Studio quantitativo e modello di simulazione finanziaria per confrontare TFS e Previdenza Complementare (TFR + Fondo Pensione Negoziale) nel Comparto Difesa e Sicurezza. Analizza 4 scenari di carriera militare tramite curve stipendiali, strategie lifecycle, tassazione agevolata e calcolo del VAN sui ritardi di erogazione dello Stato.
